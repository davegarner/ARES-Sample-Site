# About This Site

This static site scaffold is designed for **IIS 10** at the root and uses plain **HTML, CSS and JavaScript**.

## Editing model

- Update the shared header/footer partials once and all pages inherit the changes.
- Add or edit stories in `content/news/*.md` and `content/projects/*.md`.
- Change accent colors from the theme selector.
- Swap sample art and files with your production content.

## Design notes

The interface uses a neon grid aesthetic with a terminal-style modal system, centered layout and fixed-height viewport shell.

### Content hints

You can safely edit this file with headings, lists, blockquotes, links and fenced code blocks.

> The modal scrolls instead of the main page so the background shell stays locked to the viewport.
